
1. 选择排序，冒泡排序，插入排序, 快速排序，堆排序

```c++
int main()
{
    int arr[] = {2,4,9,0,1,8,7,5,6,3};
    for (int i = 0; i < 10; ++i) {
        for (int j = 1; j < 9; ++j) {
            if (arr[j] > arr[i]) {
                int tmp = arr[j];
                arr[j] = arr[i];
                arr[i] = tmp;
            }
        }
    }
}
```
